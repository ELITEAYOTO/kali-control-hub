# Contributing

Merci de contribuer à Kali Control Hub.

## Setup local
```bash
git clone https://github.com/ELITEAYOTO/kali-control-hub.git
cd kali-control-hub
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 app.py
```

## Pull Requests
- Fais une branche dédiée
- Décris clairement le changement
- Donne les étapes de test
- Garde les dépendances minimales
